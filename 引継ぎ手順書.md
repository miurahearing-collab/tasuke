1. このプロジェクトのフォルダで「Git Bash」または「コマンドプロンプト」を開く
2. 以下のコマンドを順番に実行する：

## 【よくあるエラーと対処法】

### `npm install` でエラーが出る
→ Node.jsのバージョンが古い可能性があります。Node.js v18以上をインストールしてください。

### アプリが起動しない（ポートが使われている）
```bash
# 別のポートで起動
npx vite --port 3000
```

### ログインできない
→ Firebase Authenticationにメールアドレスが登録されているか確認してください。
→ https://console.firebase.google.com/ → プロジェクト選択 → Authentication

### データが表示されない
→ ブラウザのコンソール（F12キー）でエラーメッセージを確認してください。
